# Hero Section Images

This directory contains the profile image for the hero section.

## Required Images

- `profile.png` (or .jpg, .jpeg, .webp) - Main profile photo displayed in the hero section

## Upload via Admin Panel

1. Go to `/admin` and log in
2. Navigate to "Hero Section"
3. Use the image upload field to upload your profile photo
4. The image will be automatically saved here

## Fallback

If no image is found, a purple Sparkles icon will be displayed as a fallback.
