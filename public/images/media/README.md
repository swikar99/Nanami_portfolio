# Media Section Images

This directory contains thumbnail images for videos and featured articles.

## Required Images

### Video Thumbnails
- `tedx.jpg` (or .png, .jpeg, .webp)
- `nepal-earthquake.jpg`

### Article Thumbnails
- `seplumo-interview.jpg`
- `himeberry-delicious.jpg`
- `focus-on.jpg`
- `business-insider.jpg`
- `harajuku-omotesando.jpg`
- `eris-select.jpg`
- `awlf.jpg`
- `facebook-community.jpg`

## Upload via Admin Panel

1. Go to `/admin` and log in
2. Navigate to "Media Section"
3. Each media item has an image upload field
4. Upload thumbnail images

## Fallback

- **Videos**: Show emoji thumbnail with animated gradient background
- **Articles**: Show icon (FileText/Mic) with article title
