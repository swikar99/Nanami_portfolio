# Work Section Images

This directory contains project/work card images.

## Required Images

Each project needs an image named after its `imageName` property:

- `himeberry.png` (or .jpg, .jpeg, .webp)
- `our-farms.png`
- `seplumo.png`
- `enwa-tokyo.png`
- `jobmatchy.png`
- `lernado.png`

## Upload via Admin Panel

1. Go to `/admin` and log in
2. Navigate to "Work Section"
3. Each work item has an image upload field
4. Upload images for each project

## Fallback

If no image is found, a colored gradient background with an emoji icon will be displayed.
